# ITM-03 Mid-Term: Cloud Computing Study Guide

A comprehensive interactive study guide and quiz for cloud computing fundamentals.

## 📚 Study Materials

- **Total Questions**: 55 (30 Open-ended + 25 Multiple Choice)
- **Topics Covered**: 
  - Fundamentals
  - History & Evolution
  - Architecture
  - Service Models (IaaS, PaaS, SaaS)
  - Deployment Models
  - Dynamic Infrastructure
  - Challenges & Concerns

## ✨ Features

- **Reference Tab**: Browse all questions and answers organized by category
- **Quiz Tab**: 
  - Multi-select quiz types (Open-ended, MCQ)
  - Randomize question order
  - Live scoring for MCQ questions
  - Model answers for open-ended questions
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Dark Theme**: Easy on the eyes during long study sessions
